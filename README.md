# 楽曲分析 / Music Theory Analyzer

好きな曲のコード進行・キー・BPMを楽典の観点から分析・閲覧できるWebアプリです。

🔗 **デモ**: https://<ユーザー名>.github.io/music-analyzer/

## 機能

- **収録41曲** — J-Pop / K-Pop / 洋楽 / Jazz・シティポップの定番曲
-  **コード進行表示** — 各曲のコード進行をディグリー付きで表示
-  **楽典分析** — キーとスケール、コード機能、進行のテクニックなど5項目の解説
-  **傾向分析** — 追加した曲からメジャー/マイナー比率・平均BPM・好みの傾向を診断
-  **ジャケット画像** — iTunes Search APIから自動取得
-  **AI分析（オプション）** — Anthropic APIキーを設定すると、任意の曲名を入力してAI分析が可能

## 使い方

1. デモURLを開く
2. 「追加」タブからサンプル曲を選ぶ
3. ライブラリで曲を開き「楽典・理論分析を見る」
4. 2曲以上追加すると「傾向分析」タブが解放

### AI分析機能を使う場合（任意）

1. [Anthropic Console](https://console.anthropic.com) でAPIキーを発行
2. ページ上部の入力欄にキーを貼り付け
3. 曲名を自由入力して分析

※ APIキーはブラウザのlocalStorageにのみ保存され、外部に送信されません。

## 技術構成

- 純粋なHTML / CSS / JavaScript（フレームワークなし・単一ファイル）
- ジャケット画像: iTunes Search API（キー不要）
- ホスティング: GitHub Pages

## 注意事項

- コード進行・キー情報は一般的な採譜情報および知識に基づくもので、公式楽譜とは異なる場合があります
- 「目安」と記載のある曲は特に参考程度にご覧ください
